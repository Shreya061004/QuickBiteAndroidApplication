# QuickBite - Food Ordering Android App

QuickBite is a native Android food ordering application built using Java and Android Studio. The app allows users to browse cuisines, view dishes, add items to cart, and proceed to checkout.

## Features
- Browse cuisines (Italian, Indian, Japanese, Chinese)
- View food items with images
- Add and remove items from cart
- Cart management using Singleton pattern
- RecyclerView-based dynamic lists
- Smooth activity navigation

## Tech Stack
- Java
- Android Studio
- XML Layouts
- RecyclerView
- Singleton Design Pattern
- Git & GitHub

## Project Structure
- Activities for navigation
- RecyclerView Adapter for dynamic UI
- Singleton class for cart management
- Resource-based image and layout management

## How to Run
1. Clone the repository
2. Open in Android Studio
3. Build and run on emulator or physical device
